## 1. 緊急修正（バグ・致命的UX問題）

### 1.1 ヘッダー背景アニメーション切れ問題
**問題**: Heroセクションの背景アニメーション（ParticleField、グロー効果）がヘッダー領域で途切れる
**原因**: Heroセクションの`overflow-hidden`、またはz-index/position問題
**対応**:
- [ ] `components/sections/Hero.tsx`の`overflow-hidden`を削除または調整
- [ ] 背景エフェクトをHeroセクション外（bodyレベル）に移動
- [ ] NeuralNetwork/ParticleFieldのz-indexを調整

### 1.2 ProjectModal位置バグ
**問題**: Our Productsのモーダルを開くと位置がおかしい
**対応**:
- [ ] `components/ui/dialog.tsx`のスクロール位置固定を確認
- [ ] `position: fixed`と`transform`の競合を修正
- [ ] モバイル表示でのビューポート問題を確認

### 1.3 RealtimeMetrics/DynamicIsland配置
**現状**: 右下に移動済みだが、要素が重複する可能性
**対応**:
- [ ] FloatingActions、ScrollToTop、KeyboardShortcutsHelperとの位置調整
- [ ] モバイルでの表示確認

## 3. UI/UX改善提案

### 3.1 ナビゲーション（優先度：高）
**問題点**:
- モバイルでのメニュー操作性
- スクロール時のヘッダー視認性

**改善案**:
- [ ] スティッキーヘッダーの背景をより不透明に
- [ ] モバイルメニューにアニメーション追加
- [ ] 現在セクションのハイライト表示
- [ ] スクロール進捗インジケーター（ScrollProgress）の視認性向上

### 3.2 Hero セクション（優先度：高）
**問題点**:
- CTAボタンが多すぎて迷う
- 情報の優先順位が不明確

**改善案**:
- [ ] メインCTAを1つに絞る（「プロダクトを見る」）
- [ ] サブタイトルのアニメーション改善
- [ ] TypingAnimationの速度調整
- [ ] 背景エフェクトの軽量化（パフォーマンス）

### 3.3 About セクション（優先度：中）
**改善案**:
- [ ] 統計数字をリアルな値に（設立日、開発中プロダクト数など）
- [ ] 創業者紹介セクション追加
- [ ] タイムライン形式の沿革

### 3.4 Products セクション（優先度：高）
**問題点**:
- カテゴリフィルターが2プロダクトでは不要
- モーダルUIが重い

**改善案**:
- [ ] カテゴリフィルターを削除（プロダクト数が少ない間）
- [ ] カード直接クリックで詳細展開（モーダル不要に）
- [ ] 「Coming Soon」バッジの追加
- [ ] プロダクトロゴ/アイコンの作成

### 3.5 Contact セクション（優先度：中）
**改善案**:
- [ ] フォームのバリデーションUI改善
- [ ] 送信成功/失敗のトースト通知
- [ ] reCAPTCHA導入（スパム対策）

### 3.6 Footer（優先度：低）
**改善案**:
- [ ] 不要なリンクを削除（Servicesセクションなど）
- [ ] ニュースレター登録フォーム追加（将来）
